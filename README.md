# 📱 To-Do List App - Prueba Técnica Ionic + Angular

Esta es una aplicación de lista de tareas desarrollada con **Ionic + Angular standalone**, cumpliendo los requisitos de la prueba técnica para desarrollador mobile.

## ✨ Funcionalidades principales

- ✅ Agregar, completar y eliminar tareas.
- 🗂 Gestión de categorías (crear, eliminar, asignar a tareas, filtrar).
- 🔥 Integración con Firebase y Remote Config (feature flags).
- 💾 Almacenamiento local de tareas y categorías.
- 📱 Preparada para compilar en Android e iOS usando **Cordova**.
- 🚀 Optimizada para rendimiento y carga rápida.

---

## ⚙️ Requisitos

- Node.js ≥ 18
- Ionic CLI: `npm install -g @ionic/cli`
- Cordova CLI: `npm install -g cordova`
- Java JDK 8 o superior
- Android SDK
- (Opcional para iOS) macOS + Xcode
- Firebase cuenta gratuita (para Remote Config)
